# Roblox 2016 Web
Welcome to the classic 2016 Roblox experience. 
This project is dedicated to preserving the UI/UX style of the 2016 era.

## Features
- Classic 2016-inspired layout.
- Lightweight and fast loading.
- Easy to manage games via JSON.

## How to manage
1. Add your games to `data/games.json`.
2. Keep your assets organized in the `/assets/` directory.
3. Don't delete system files or the repository might lose its structure!
